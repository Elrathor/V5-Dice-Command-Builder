Dice Command Builder

  • Example: If you have to roll Wits + Awareness use the number of points of Wits as Stat 1 and the number of points of Awareness as Stat 2. Any further modification can be entered as Modifier.
  • The hunger points and the number of successes needed are only used when activated. (See checkbox in front of input) A deactivated value defaults to 0.
  • Hint: In case a flat number of D10 should be rolled you can just enter the number as a modifier and leave everything else at 0.
  • The command at the bottom should (if no error is displayed) be a reasonable input for the V5 function of the Vampire: The Masquerade Discord Dice Bot made by Stonewolf18.

{{ stat1 }}
{{ stat2 }}
Hunger: {{ hunger }}
Successes: {{ successNeeded }}
